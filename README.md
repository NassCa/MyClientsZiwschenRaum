# MyClients — Nass

Praxissoftware für Psychotherapeutinnen

## Beschreibung
MyClients ist eine lokale, DSGVO-konforme Praxisverwaltungs-App für Psychotherapeutinnen.  
Alle Daten werden ausschließlich lokal im Browser gespeichert (localStorage).  
Keine Serververbindung, keine Datenübertragung.

## Design
Farbschema: **Grün (#4d7260)**

## Installation (GitHub Pages)
1. Repository auf GitHub anlegen
2. Alle Dateien ins Root hochladen
3. Settings → Pages → Branch: main → / (root)
4. App unter `https://[username].github.io/MyClientsNass/` aufrufen
5. Auf dem Gerät: „Zum Homescreen hinzufügen" für PWA-Installation

## PWA-Features
- Offline-fähig (Service Worker)
- Installierbar auf iOS & Android
- Kein App-Store nötig
